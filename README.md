# Projeto Portfólio Pessoal - Web Development

Este projeto consiste num portfólio pessoal desenvolvido como atividade prática para a disciplina de Programação. O objetivo foi aplicar conceitos fundamentais de HTML5, CSS3 (Bootstrap) e JavaScript/jQuery.

## Integrantes
* **Miguel Pereira da Rocha Silva** - 4º Período de Sistemas de Informação (UPE)

## Descrição do Site
O site é composto por 5 páginas interligadas:
1. **Home (index.html):** Apresentação inicial com foto e frase de impacto.
2. **Sobre (sobre.html):** Detalhes sobre a trajetória acadêmica e objetivos profissionais.
3. **Habilidades (habilidades.html):** Listagem de competências técnicas e soft skills.
4. **Cursos (cursos.html):** Tabela com formações concluídas e em andamento.
5. **Contato (contato.html):** Formulário funcional com validação via JavaScript.

## Tecnologias Utilizadas
* **HTML5:** Estrutura semântica das páginas.
* **CSS3 / Bootstrap 5:** Estilização, layout em Grid/Flexbox e responsividade para dispositivos móveis.
* **JavaScript:** * Validação de campos no formulário de contacto.
    * Menu de navegação responsivo (hambúrguer).
    * Funcionalidade de alternância de tema (Dark Mode).

## Como Executar o Projeto Localmente
1. Faça o download ou clone este repositório.
2. Certifique-se de que todos os arquivos (.html, .css, .js e as imagens) estão na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador web (Chrome, Firefox, Edge, etc.).
